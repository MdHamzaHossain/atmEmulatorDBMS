# ATM Emulator
A full stack app emulating a barebones site with a database. 

## How To Install
- Clone the repository
- `npm install` 
- Create `db/atm.db`
- Run `node .`

## Tech
- `ExpressJS` backend server
- `Sqlite3` local file Database
- Raw HTML-CSS-JS barebone server
